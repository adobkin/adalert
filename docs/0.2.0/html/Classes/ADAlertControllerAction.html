Overview

The ADAlertControllerAction instance represents an action. For each action ADAlertController creates and configures a button. When the user taps that button, the alert controller executes the block wich you specified when creating the action instance.

You should use this class to add information about an action, including the title and style of button, and a handler to execute when the user taps the button. After creating the action, add it to the ADAlertController instance before displaying it.
